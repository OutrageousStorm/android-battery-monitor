# 🔋 Android Battery Monitor

Real-time battery drain monitor running as an Android app via Shizuku.

## Features
- Top 10 battery hogs (by CPU time + wakelock)
- Per-process breakdown (foreground vs background)
- Live refresh every 2 seconds
- No root needed (uses Shizuku)
- Minimal overhead

## Install
1. Install Shizuku via Magisk/KernelSU
2. Grant permission to this app
3. Open and monitor
